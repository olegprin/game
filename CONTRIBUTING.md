<p>This is a gem that simply includes jQuery and jQuery UJS into the Rails
asset pipeline. The asset pipeline functionality is well tested within the
Rails framework. And jQuery and jQuery UJS each have their own extensive
test suites. Thus, there's not a lot to actually test here.</p>

<p>That being said, we do have a few integration-level tests to make sure
everything is being included and basic UJS functionality works within a
sample Rails app.</p>

<p>If you're making changes to the actual gem, run the tests as follows:</p>

<ol>
<li><p>Checkout the demo Rails app: <code>git clone git://github.com/JangoSteve/Rails-jQuery-Demo.git</code></p></li>
<li><p>Install the gems: <code>bundle install</code></p></li>
<li><p>Change the jquery-rails gem in the Gemfile to use your local
version of the gem with your updates: <code>gem 'jquery-rails', :path =&gt; '../path/to/jquery-rails'</code></p></li>
<li><p>Update your bundled jquery-rails gem: <code>bundle update jquery-rails</code></p></li>
<li><p>Run the tests: <code>bundle exec rspec spec/</code></p></li>
</ol>

<p>We definitely appreciate pull requests that highlight or reproduce a problem, even without a fix.</p>
